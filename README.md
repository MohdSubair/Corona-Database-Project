# 🦠 Corona Virus Data Analysis

A data analysis project that explores the global COVID-19 (Coronavirus) pandemic using Python. This project demonstrates data cleaning, preprocessing, visualization, and exploratory data analysis (EDA) to identify trends and insights from COVID-19 datasets.

## 📌 Project Overview

The objective of this project is to analyze COVID-19 data and gain insights into:

- Total confirmed cases
- Total deaths
- Total recovered cases
- Active cases
- Country-wise analysis
- Trend visualization
- Statistical summaries

This project is suitable for beginners learning Data Analysis using Python.

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/Corona-Virus-Data-Analysis.git
```

2. Navigate to the project

```bash
cd Corona-Virus-Data-Analysis
```

3. Install required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

4. Open the notebook or HTML file

```bash
jupyter notebook
```

or simply open

```
corana.html
```

in your web browser.
